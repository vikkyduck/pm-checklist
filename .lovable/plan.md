

## Куда нажимать в текущем экране

На скриншоте вы находитесь в **Cloud → Users → Authentication settings**. Раздел SMTP спрятан внизу под кнопкой **Advanced** — её видно в самом низу справа на вашем скриншоте.

### Пошаговый маршрут

1. **Прокрутите правую панель вниз** до конца — там есть кнопка `Advanced ›`
2. **Нажмите `Advanced`** — раскроются дополнительные настройки
3. **Найдите блок SMTP** — это раздел с полями для ввода Host, Port, User, Password, Sender email, Sender name
4. **Заполните поля** следующим образом:

| Поле | Значение |
|---|---|
| Host | `smtp.yandex.ru` |
| Port | `465` |
| User | `noreply@vi-utkina.ru` |
| Password | ваш пароль приложения из Яндекса |
| Sender email | `noreply@vi-utkina.ru` |
| Sender name | `PM Roadmap` |

5. **Сохраните** настройки

### Важно

- Если раздел Advanced не раскрывается или SMTP-полей там нет — напишите мне, разберёмся
- После сохранения SMTP попробуйте войти через магическую ссылку на тестовом email — письмо должно прийти от `noreply@vi-utkina.ru`
- Рекомендую **пересоздать пароль приложения** в Яндексе, так как он был виден в чате ранее — это вопрос безопасности
